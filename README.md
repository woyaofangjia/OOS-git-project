# 校园二手交易平台

## 项目概述

基于Java技术栈开发的C2C校园二手交易平台，旨在为在校学生提供安全、便捷的二手商品交易环境。平台采用SSM架构，实现了商品发布、求购信息发布、购物车管理、个人信息管理等核心功能，支持多用户并发访问，具备良好的安全性和性能表现。

## 技术栈

### 核心技术
- **后端框架**：Spring 5.3.23 + Spring MVC 5.3.23 + MyBatis 3.5.10 (SSM)
- **开发语言**：Java 21
- **数据库**：MySQL 8.0.40
- **前端技术**：HTML5 + CSS3 + JavaScript (ES5/ES6) + jQuery 3.6.4 + EasyUI 1.10.12

### 辅助技术
- **模板引擎**：Thymeleaf 3.0.15
- **通信协议**：WebSocket 1.1
- **安全机制**：MD5加密、Token验证、XSS/CSRF防护
- **工具技术**：敏感词过滤、图片压缩处理

## 系统架构与模块设计

### 整体架构
采用经典的MVC三层架构：
- **表现层**：Thymeleaf模板 + EasyUI组件库，实现页面渲染和用户交互
- **业务逻辑层**：Spring管理的Service组件，封装核心业务规则
- **数据访问层**：MyBatis映射，实现数据库操作的封装与优化

### 核心模块

| 模块 | 主要功能 | 技术亮点 |
|------|----------|----------|
| **用户模块** | 注册、登录、个人信息管理 | MD5密码加密、Token验证 |
| **商品模块** | 商品发布、分类展示、详情查看、图片上传 | 三级分类联动、图片压缩优化 |
| **求购模块** | 求购信息发布、浏览 | 分类筛选、时间排序 |
| **交易模块** | 购物车管理、订单生成、交易状态跟踪 | 订单数据管理、交易流程控制 |
| **搜索模块** | 商品/求购信息搜索、筛选排序 | 模糊查询、分类筛选 |
| **消息模块** | 商品留言、求购留言 | 敏感词过滤、留言管理 |

## 核心功能实现

### 1. 单用户登录系统
- **实现方式**：基于WebSocket和Spring WebSocket实现实时会话管理
- **技术难点**：解决多端登录冲突、确保会话状态一致性
- **解决方案**：使用线程安全的Map存储用户登录状态，通过时间戳比较实现登录踢人功能

### 2. 商品发布与管理
- **实现方式**：三级分类联动选择、图片上传与预览、表单验证
- **技术难点**：大文件上传优化、图片格式验证与压缩
- **解决方案**：实现图片上传与压缩，支持多图片上传与预览

### 3. 购物车功能
- **实现方式**：基于Cookie和数据库的双层存储机制
- **技术难点**：购物车数据同步、并发更新冲突
- **解决方案**：使用乐观锁机制处理并发更新，实现登录前后购物车数据自动合并

### 4. 站内搜索引擎
- **实现方式**：基于MyBatis的动态SQL和数据库索引优化
- **技术难点**：搜索性能优化、结果排序准确性
- **解决方案**：建立商品名称、描述等字段的联合索引，实现按匹配度和时间的综合排序

## 技术亮点与性能优化

### 1. 安全性设计
- **数据加密**：密码使用MD5加密存储
- **防攻击措施**：实现SQL注入防护、CSRF令牌验证
- **权限控制**：实现基本的角色权限管理，区分普通用户和管理员

### 2. 性能优化
- **数据库优化**：合理设计索引，使用分页查询减少数据传输量
- **图片优化**：实现图片压缩处理，减少存储空间和加载时间

### 3. 代码质量保障
- **设计模式**：采用单例模式优化工具类设计
- **异常处理**：实现基本的异常处理机制
- **代码规范**：遵循Java编码规范，使用Maven进行依赖管理和构建

## 开发环境与快速启动

### 环境要求
- JDK 21+  
- Maven 3.9.x+  
- MySQL 8.0.x+  
- Jetty 9.4.x+ (Maven插件提供)

### 快速启动

1. **配置数据库**
   ```sql
   -- 创建数据库
   CREATE DATABASE `Campus-SecondHand` CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
   ```bash
   -- 导入数据结构 (Windows环境)
   mysql -u root -p Campus-SecondHand < SQL\校园二手交易平台数据库1.0.sql
   ```

2. **修改配置**
   - 编辑 `src/main/resources/jdbc.properties` 配置数据库连接
   - 确保使用正确的MySQL 8.0驱动类：`com.mysql.cj.jdbc.Driver`

3. **启动项目**
   ```bash
   # 进入项目根目录
   cd d:\code\Campus-SecondHand
   # 使用Maven启动Jetty服务器
   mvn jetty:run
   ```

4. **访问平台**
   - 浏览器访问：`http://localhost:8080`
   - 管理员账号：`admin` / `123456`
